## 공식문서

[Vue.js](vuejs.org)

api 문서 찾아보고 적용해보자.

## 설치

`[Vue CLI 2.x]`

vue init '프로젝트 템플릿 유형` `프로젝트 폴더 위치`
vue init webpack-simple '프로젝트 폴더 위치'

`[Vue CLI 3.x]`
vue create '프로젝트 폴더 위치'

ex) vue create vue-cli

vue cli 4.x 버전은

vue2를 선택

## 실행

cd 프로젝트 폴더위치
npm run serve
